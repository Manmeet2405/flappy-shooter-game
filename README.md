# flappy-shooter-game
🎮 Flappy Shooter - Python Game

A modern twist on the classic Flappy Bird game, built using Python and Pygame, featuring shooting mechanics, enemies, and advanced visual effects.

🚀 Features
🐦 Smooth Flappy Bird-style movement with gravity physics
🔫 Shooting system with bullets and cooldown
👾 Multiple enemy types (wave movement & chasing AI)
💥 Particle effects, explosions, and screen shake
🌌 Animated background with starfield
🎯 Score system with increasing difficulty
❤️ Health system with damage mechanics
🎨 Beautiful UI with menus, settings, and credits
🔊 Sound effects (generated programmatically)


🎮 Controls
SPACE → Flap / Jump
F / Right Shift / Mouse Click → Shoot
ENTER → Restart after Game Over
SPACE → Return to Menu


🛠️ Installation & Setup
-Clone the repository
-Navigate to the folder
-cd flappy-shooter-game
Install dependencies:
-pip install pygame
Run the game:
-python game.py


📦 Optional: Run EXE Version
If included:
Go to /dist folder
Run FlappyShooter.exe
